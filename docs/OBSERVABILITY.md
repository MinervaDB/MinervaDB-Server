## Monitoring Architecture

```
ClickHouse Nodes
     │
          ├── /metrics (Prometheus format)
               ├── system.metrics
                    ├── system.asynchronous_metrics
                         ├── system.events
                              └── Server Logs (/var/log/clickhouse-server/)
                                         │
                                             ┌──────▼───────────────────────────────────────────┐
                                                 │              Observability Pipeline               │
                                                     │                                                   │
                                                         │  clickhouse_exporter ──► Prometheus ──► Grafana  │
                                                             │                                                   │
                                                                 │  Vector (logs) ──────► Loki / Elasticsearch      │
                                                                     │                                                   │
                                                                         │  OTLP traces ────────► Jaeger / Tempo            │
                                                                             └──────────────────────────────────────────────────┘
                                                                             ```

                                                                                                                                                                                                                   ## Key System Tables
                                                                                                                                                                                                                   
                                                                                                                                                                                                                   ### Active Queries
                                                                                                                                                                                                                   
                                                                                                                                                                                                                   ```sql
                                                                                                                                                                                                                   SELECT
                                                                                                                                                                                                                       query_id,
                                                                                                                                                                                                                           user,
                                                                                                                                                                                                                               elapsed,
                                                                                                                                                                                                                                   formatReadableSize(memory_usage) AS memory,
                                                                                                                                                                                                                                       formatReadableSize(read_bytes) AS read_bytes,
                                                                                                                                                                                                                                           read_rows,
                                                                                                                                                                                                                                               total_rows_approx,
                                                                                                                                                                                                                                                   left(query, 100) AS query_preview
                                                                                                                                                                                                                                                   FROM system.processes
                                                                                                                                                                                                                                                   ORDER BY elapsed DESC;
                                                                                                                                                                                                                                                   ```
                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                   ### Query History Analysis
                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                   ```sql
                                                                                                                                                                                                                                                   -- Slowest queries in last hour
                                                                                                                                                                                                                                                   SELECT
                                                                                                                                                                                                                                                       query_id,
                                                                                                                                                                                                                                                           user,
                                                                                                                                                                                                                                                               event_time,
                                                                                                                                                                                                                                                                   round(query_duration_ms / 1000, 2) AS duration_sec,
                                                                                                                                                                                                                                                                       formatReadableSize(memory_usage) AS peak_memory,
                                                                                                                                                                                                                                                                           read_rows,
                                                                                                                                                                                                                                                                               formatReadableSize(read_bytes) AS data_read,
                                                                                                                                                                                                                                                                                   exception,
                                                                                                                                                                                                                                                                                       normalizeQuery(query) AS query_template
                                                                                                                                                                                                                                                                                       FROM system.query_log
                                                                                                                                                                                                                                                                                       WHERE
                                                                                                                                                                                                                                                                                           type = 'QueryFinish'
                                                                                                                                                                                                                                                                                               AND event_time >= now() - INTERVAL 1 HOUR
                                                                                                                                                                                                                                                                                               ORDER BY query_duration_ms DESC
                                                                                                                                                                                                                                                                                               LIMIT 20;
                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                               -- Error rate by user
                                                                                                                                                                                                                                                                                               SELECT
                                                                                                                                                                                                                                                                                                   user,
                                                                                                                                                                                                                                                                                                       countIf(type = 'ExceptionBeforeStart') AS errors_before_start,
                                                                                                                                                                                                                                                                                                           countIf(type = 'ExceptionWhileProcessing') AS errors_while_processing,
                                                                                                                                                                                                                                                                                                               countIf(type = 'QueryFinish') AS successful,
                                                                                                                                                                                                                                                                                                                   round(countIf(type != 'QueryFinish') * 100.0 / count(), 2) AS error_pct
                                                                                                                                                                                                                                                                                                                   FROM system.query_log
                                                                                                                                                                                                                                                                                                                   WHERE event_time >= now() - INTERVAL 24 HOUR
                                                                                                                                                                                                                                                                                                                   GROUP BY user
                                                                                                                                                                                                                                                                                                                   ORDER BY error_pct DESC;
                                                                                                                                                                                                                                                                                                                   ```
                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                   ### Replication Health
                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                   ```sql
                                                                                                                                                                                                                                                                                                                   -- Replica lag overview
                                                                                                                                                                                                                                                                                                                   SELECT
                                                                                                                                                                                                                                                                                                                       hostName() AS host,
                                                                                                                                                                                                                                                                                                                           database,
                                                                                                                                                                                                                                                                                                                               table,
                                                                                                                                                                                                                                                                                                                                   is_leader,
                                                                                                                                                                                                                                                                                                                                       is_readonly,
                                                                                                                                                                                                                                                                                                                                           absolute_delay,
                                                                                                                                                                                                                                                                                                                                               queue_size,
                                                                                                                                                                                                                                                                                                                                                   inserts_in_queue,
                                                                                                                                                                                                                                                                                                                                                       merges_in_queue
                                                                                                                                                                                                                                                                                                                                                       FROM system.replicas
                                                                                                                                                                                                                                                                                                                                                       WHERE absolute_delay > 0 OR queue_size > 0
                                                                                                                                                                                                                                                                                                                                                       ORDER BY absolute_delay DESC;
                                                                                                                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                                                                       -- Replication queue errors
                                                                                                                                                                                                                                                                                                                                                       SELECT
                                                                                                                                                                                                                                                                                                                                                           database,
                                                                                                                                                                                                                                                                                                                                                               table,
                                                                                                                                                                                                                                                                                                                                                                   type,
                                                                                                                                                                                                                                                                                                                                                                       last_exception,
                                                                                                                                                                                                                                                                                                                                                                           num_tries,
                                                                                                                                                                                                                                                                                                                                                                               postpone_reason
                                                                                                                                                                                                                                                                                                                                                                               FROM system.replication_queue
                                                                                                                                                                                                                                                                                                                                                                               WHERE last_exception != ''
                                                                                                                                                                                                                                                                                                                                                                               ORDER BY num_tries DESC;
                                                                                                                                                                                                                                                                                                                                                                               ```
                                                                                                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                                                                                               ### Merge and Mutation Monitoring
                                                                                                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                                                                                               ```sql
                                                                                                                                                                                                                                                                                                                                                                               -- Active merges
                                                                                                                                                                                                                                                                                                                                                                               SELECT
                                                                                                                                                                                                                                                                                                                                                                                   database,
                                                                                                                                                                                                                                                                                                                                                                                       table,
                                                                                                                                                                                                                                                                                                                                                                                           round(elapsed, 2) AS elapsed_sec,
                                                                                                                                                                                                                                                                                                                                                                                               round(progress * 100, 2) AS pct_complete,
                                                                                                                                                                                                                                                                                                                                                                                                   num_parts,
                                                                                                                                                                                                                                                                                                                                                                                                       source_part_names,
                                                                                                                                                                                                                                                                                                                                                                                                           result_part_name,
                                                                                                                                                                                                                                                                                                                                                                                                               formatReadableSize(total_size_bytes_compressed) AS size_compressed,
                                                                                                                                                                                                                                                                                                                                                                                                                   merge_type
                                                                                                                                                                                                                                                                                                                                                                                                                   FROM system.merges
                                                                                                                                                                                                                                                                                                                                                                                                                   ORDER BY elapsed_sec DESC;
                                                                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                                                                   -- Pending mutations
                                                                                                                                                                                                                                                                                                                                                                                                                   SELECT
                                                                                                                                                                                                                                                                                                                                                                                                                       database,
                                                                                                                                                                                                                                                                                                                                                                                                                           table,
                                                                                                                                                                                                                                                                                                                                                                                                                               mutation_id,
                                                                                                                                                                                                                                                                                                                                                                                                                                   command,
                                                                                                                                                                                                                                                                                                                                                                                                                                       create_time,
                                                                                                                                                                                                                                                                                                                                                                                                                                           parts_to_do,
                                                                                                                                                                                                                                                                                                                                                                                                                                               is_done,
                                                                                                                                                                                                                                                                                                                                                                                                                                                   latest_failed_part,
                                                                                                                                                                                                                                                                                                                                                                                                                                                       latest_fail_reason
                                                                                                                                                                                                                                                                                                                                                                                                                                                       FROM system.mutations
                                                                                                                                                                                                                                                                                                                                                                                                                                                       WHERE is_done = 0
                                                                                                                                                                                                                                                                                                                                                                                                                                                       ORDER BY create_time;
                                                                                                                                                                                                                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                                                                                                                                                                       -- Part count (watch for "too many parts" warnings)
                                                                                                                                                                                                                                                                                                                                                                                                                                                       SELECT
                                                                                                                                                                                                                                                                                                                                                                                                                                                           database,
                                                                                                                                                                                                                                                                                                                                                                                                                                                               table,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                   count() AS active_parts,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                       sum(rows) AS total_rows,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           formatReadableSize(sum(bytes_on_disk)) AS disk_usage
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           FROM system.parts
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           WHERE active = 1
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           GROUP BY database, table
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           HAVING active_parts > 100
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           ORDER BY active_parts DESC;
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           ```
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           ### Disk and Storage
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           ```sql
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           -- Disk usage summary
                                                                                                                                                                                                                                                                                                                                                                                                                                                                           SELECT
                                                                                                                                                                                                                                                                                                                                                                                                                                                                               name AS disk_name,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   path,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       formatReadableSize(free_space) AS free,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           formatReadableSize(total_space) AS total,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               round((1 - free_space / total_space) * 100, 1) AS used_pct,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   type
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   FROM system.disks
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   ORDER BY used_pct DESC;
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   -- Tiered storage policy status
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   SELECT
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       database,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           table,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               formatReadableSize(sum(bytes_on_disk)) AS size,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   count() AS parts,
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       disk_name
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       FROM system.parts
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       WHERE active = 1
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       GROUP BY database, table, disk_name
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       ORDER BY database, table, disk_name;
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       ```
